# 📒 Contact Book (Python)

A simple **Contact Book application** built using **Python** and **list of dictionaries**.
This program allows users to **add, view, search, and delete contacts** through a menu-driven command line interface.

---

## 🚀 Features

* ➕ **Add Contact** – Store name, phone number, and email
* 📋 **View Contacts** – Display all saved contacts
* 🔍 **Search Contact** – Find a contact by name
* 🗑 **Delete Contact** – Remove a contact from the list
* ❌ **Exit Program**

---

## ⚙️ How It Works

Contacts are stored using a **list of dictionaries**.

Example:

```python
contacts = [
    {
        "name": "John",
        "phone": "9876543210",
        "email": "john@email.com"
    }
]
```

Each contact contains:

* **Name**
* **Phone Number**
* **Email**

## ▶️ How to Run the Program

1. Install **Python** (if not installed).
2. Clone this repository:

```
git clone https://github.com/your-username/contact-book.git
```

3. Navigate to the project folder:

```
cd contact-book
```

4. Run the program:

```
python contact_book.py
```
